#### Initialize frontend in repository
```bash
npm create vite@latest frontend
```

#### Configure AWS Profile (Access Credentials from AWS Console or downloaded CSV)
```bash
aws configure --profile free-app-ci
```

#### Verify AWS Account Config
```bash
aws sts get-caller-identity --profile free-app-ci
```

#### Deploy Stack
```bash
aws cloudformation deploy \
  --template-file infra/frontend-s3-cloudfront.yml \
  --stack-name free-app-frontend \
  --parameter-overrides BucketName=free-app-frontend \
  --capabilities CAPABILITY_NAMED_IAM \
  --profile free-app-ci \
  --no-fail-on-empty-changeset
```

#### Get Deployment Status
```bash
aws cloudformation describe-stacks \
  --stack-name free-app-frontend \
  --profile free-app-ci \
  --query "Stacks[0].StackStatus"
```

#### Get Stack Outputs (Cloudfront URL)
```bash
aws cloudformation describe-stacks \   
  --stack-name free-app-frontend \
  --profile free-app-ci \
  --query "Stacks[0].Outputs"
```

#### Delete Stack
```bash
aws cloudformation delete-stack \
  --stack-name free-app-frontend \ 
  --profile free-app-ci
```
